STEP 01

실행까지 가는 길

이탈 없이 한 번에 성공해 봅시다.
아래 순서대로 진행해 주세요.

1️⃣ 파일 확인

다운받은 파일명과 용량을 확인해주세요.
(파일명: roosycozy.exe 또는 .zip)

2️⃣ 압축 풀고 실행

반드시 압축을 풀고 실행하세요.
Mac 유저는 응용프로그램 폴더로 옮겨도 좋습니다.

3️⃣ 경고창 무시 (중요!)

'서명되지 않은 앱' 경고는 베타 버전에선 정상입니다.
[추가 정보] → [실행]을 눌러주세요.

4️⃣ 살짝 기다리기

첫 실행은 10~30초 정도 걸립니다.
멈춘 것 같아도 조금만 기다려주세요.

🛟 혹시 실행이 막히나요?

Windows
추가 정보 → 실행
Mac
우클릭 → 열기
